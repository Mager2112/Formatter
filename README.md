# Formatter
## Основы
**Formatter** — программа, позволяющая анализировать файлы и создавать из них таблицы. В таблицах содержатся данные об имени человека, его возрасте, адресе и дате рождения. Реализовано несколько команд, позволяющих пользователю взаимодействовать с программой.

> [!Note]
> python formatter.py -i <файл>
>
> python formatter.py --input <файл>
>
> python formatter.py -i <путь к файлу>
>
> python formatter.py --input <ссылка>

> [!Tip]
> Можно также ввести команду python formatter.py --help для дополнительных сведений

## Требования
1. ✅Вывести в консоль таблицу таким образом, чтобы все данные отображались корректно 
2. ✅У таблицы должен быть заголовок
3. ✅У столбцов таблицы должен быть заголовок
4. ✅Столбцы должны быть ровными, для разделения можно использовать символ |
5. ✅Дата рождения была в одном формате у всех людей
6. ✅Формат даты в таблице должен быть приведен к виду YYYY-MM-DD HH:MM:SS
7. ✅Скрипт должен корректно работать в linux и windows
8. ✅Учитывать ширину консоли. Таблица должна быть во всю ширину экрана. Если вся информация по какой-то причине не влезает в ширину консоли - программа должна сокращать отображение данных каким-либо образом
9. ✅Программа должна уметь обрабатывать исходный файл в любой кодировке
10. ✅Исходный текстовый файл может иметь любой размер вплоть до 1GB
11. ✅Сделать так чтобы программа могла принимать на вход абсолютный или относительный путь до файла именованным параметром. Например:
> python script.py -i <любое_название>.txt
> 
> python script.py --input <любое_название>.txt
12. ✅Сделать так чтобы программа могла принимать на вход ссылку на файл именованным параметром. Например:
>python script.py -i https://gitlab.com/toptenov/automation_qa_intern_2026/-/raw/main/data.txt
>
> python script.py --input https://gitlab.com/toptenov/automation_qa_intern_2026/-/raw/main/data.txt
13. ✅Покрыть данный проект тестами
